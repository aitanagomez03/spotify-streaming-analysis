# spotify-streaming-analysis

🎧 Streaming Analysis of My Spotify Listening History
This project is a data‑driven exploration of my personal Spotify streaming habits. Using Python, the Spotify API, and interactive visualizations, I analyze how my music preferences evolve over time and what they reveal about my listening behavior.
🌟 Project Overview
The goal of this project is to transform raw Spotify data into meaningful insights. By combining descriptive analytics, audio‑feature exploration, and visual storytelling, the project highlights patterns in my listening activity, my top artists and tracks, and the characteristics of the music I gravitate toward.
🎯 Objectives
- Identify long‑term listening trends and seasonal patterns
- Explore my most‑played artists, tracks, and genres
- Analyze audio features such as energy, valence, danceability, and tempo
- Visualize listening behavior across days, months, and hours
- Practice data cleaning, transformation, and visualization techniques
- Experiment with the Spotify Web API and personal streaming data
🛠️ Technologies Used
- Python: pandas, numpy, matplotlib, seaborn, plotly
- Spotify Web API
- Jupyter Notebook
- (Optional) Streamlit / Tableau / Power BI for dashboards
📦 Data Sources
The dataset comes from two possible sources:
- Spotify Personal Data Export (“Your Data” package), which includes JSON files with detailed streaming history
- Spotify Web API, used to retrieve metadata and audio features for tracks, artists, and albums
No private credentials or personal information are included in this repository.
📊 Main Analyses
- Total minutes streamed per month and per year
- Top artists, tracks, and genres
- Listening frequency by hour of the day, day of the week and month
📈 Visualizations
The project includes a variety of visual outputs, such as:
- Bar charts of time of day, day of the week and month where I listen most music
🧠 Key Insights
Some of the most interesting findings include:
- Clear patterns in the times of day, day of the week and month when I listen to music the most
- My top artists and tracks
